Concept

When a responsive navbar gets to mobile size the usually thing that happen is the links collapse and they slide out in list form.

Most CSS frameworks animate height using transitions, this can be bad on mobile seeing as the height property causes the browser to re-paint. Using CSS transforms are great on all device because the browser loads them on the gpu.

When the sidebar opens the page content also moves. All the content is within a container which has overflow hidden to prevent scrolling.

The content overlay on the content fires the same function as the close button (NOTE: Content overlay is coloured only for demo).
